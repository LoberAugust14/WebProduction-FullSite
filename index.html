HOW TO INSTALL A PHP-BASED WEB APPLICATION

  • 1. First the user must create an Amazon Web Services (AWS) account.
  • 2. Once, the user has created the AWS account, they must access the AWS Lightsail page.
  • 3. From the AWS Lightsail page, the user must create an instannce. When creating an instance the user must select from a blueprint.

    • Photo via AWS, Building a Photo Diary on Amazon Lightsail with Ghost BY Emma White

  • 4. Once a blueprint is selected the user will scroll to the bottom of the page, agree to a payment plan and click the "Create instance" button.
  • 5. Once the instance is created, the user is given a username, an IP address and an IPv6 address.
  • 6. When ready to bein making the website, the user will click on the "Conect using SSH" button.
  • 7. This button will connect the user to an SSH Terminal called Bitnami.
  • 8. In Bitnami the user must us the basic code of the website to uncover the password given to them by Bitnami. It is extremely important to remember this password because it is only given once.The user will need this password to connect to a database and eventually to their desired PHP site such as Joomla or WordPress.
  • 9. The user will use the username and password they were given by Bitnami, to create their own.
  • 10. Once the user has created their custom username and password, they will conncect to the desired PHP-based web application of their choice.
  • 11. Once connected to the log in page, the user will use the username and password they created to login on their desired PHP-based web application.
  • 12. From there the user is free to use the all the features of their desired PHP-based web application.
